# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-01-05

### Added
- Initial release of PicoDSP.
- Virtual Analog Engine based on `infinitedsp-core`.
- USB Audio Class 1.0 support (48kHz Stereo).
- USB MIDI support with Note On/Off and CC control.
- Dual-core architecture (Core 0: USB, Core 1: DSP).
- Flash storage for presets with SysEx dump/restore support.
- Effects chain: Delay, Reverb, Stereo Widener.
